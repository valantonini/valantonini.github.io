DbUp Serilog Upgrade Log

A DbUp Upgrade Log that logs to serilog. You can view the project at https://github.com/valantonini/DbUp.SerilogUpgradeLog or install via nuget

Install-Package DbUp.SerilogUpgradeLogger

Usage:

var upgrader = DeployChanges.To
                            .SqlDatabase("")
                            .WithScriptsEmbeddedInAssembly(Assembly.GetExecutingAssembly())
                            .LogToConsole()
                            .LogToSerilog() //add this line
                            .Build();
